# BiblioBloom
데이터 베이스 3분반 6조 정승원, 정지우
---

## 요구 사항
이 프로젝트를 실행하기 전에 다음 소프트웨어가 설치되어 있어야 합니다:
- **Python 3.9+**
- **Node.js 16+**
- **MySQL 8.0+**
- **npm** 또는 **yarn** (Node.js 설치 시 포함)

---
